About form-filler

Learn more about this Angular + Tailwind CSS starter template.

This starter template provides a solid foundation for building modern web applications with Angular and Tailwind CSS. It includes everything you need to get started quickly while following best practices for maintainable, scalable code.

Key Features:

  • Angular Standalone APIs: Modern component architecture
  • Tailwind CSS: Utility-first styling with custom design system
  • Dark Mode: System preference detection and manual toggle
  • Responsive Design: Mobile-first, adaptive layouts
  • TypeScript: Full type safety and modern JavaScript features
  • Code Quality: ESLint, Prettier, and Husky pre-commit hooks
  • Testing: Playwright for reliable end-to-end testing
  • PWA Ready: Optional Progressive Web App features
  • • Angular: Latest version with standalone components
  • • Tailwind CSS: Utility-first CSS framework
  • • TypeScript: Strongly typed JavaScript
  • • Playwright: Modern E2E testing framework
  • • ESLint & Prettier: Code quality and formatting

To start development:

  1. Run npm install
  2. Start dev server with ng serve
  3. Open http://localhost:4200
